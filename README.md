# Casa do Avivamento

Projeto desenvolvido para a igreja local **Casa do Avivamento**, com o objetivo de fortalecer sua presença digital através de um site moderno, responsivo e acessível.

A plataforma foi criada pensando em proporcionar uma experiência agradável tanto para membros quanto para visitantes, apresentando informações da igreja de forma simples, organizada e intuitiva.

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- HTML5
- CSS3

# 🎯 Objetivo do projeto

O principal objetivo deste projeto foi desenvolver um site institucional moderno para a igreja, permitindo que visitantes e membros possam:

- Conhecer a igreja
- Visualizar horários de cultos
- Acompanhar eventos e programações
- Encontrar informações de contato
- Navegar facilmente em qualquer dispositivo

---

# ⚙️ Instalação e execução

## Clone o repositório

```bash
git clone https://github.com/marcelodevv/casadoavivamento.git
```

## Acesse a pasta do projeto

```bash
cd casadoavivamento
```

## Execute o projeto

Basta abrir o arquivo:

```bash
index.html
```

Ou utilize a extensão **Live Server** no VSCode.
